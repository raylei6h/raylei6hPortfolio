# Assets Folder

This folder contains all media assets for the portfolio.

## Structure

```
assets/
├── images/           # Project screenshots and images
└── CV_ILAG-2025.pdf # Your resume/CV file
```

## Required Images

Place the following images in the `images/` folder:

### Project Images (Recommended size: 1920x1080 or 16:9 aspect ratio)
- `pet-feeder.jpg` - IoT Smart Pet Feeder System
- `uniqos.jpg` - UniqOS Operating System
- `pizza-system.jpg` - Pizza Ordering System
- `employee-record.jpg` - Employee Record System
- `hotel-booking.jpg` - Hotel Booking System
- `library-system.jpg` - Library Management System
- `placeholder.jpg` - Fallback image for missing projects

### Optional Additional Images
- `profile.jpg` - Your professional photo (for About section)
- `hero-bg.jpg` - Custom hero background
- `certifications/` - Certification badge images

## Image Guidelines

### Format
- Use **JPG** for photographs
- Use **PNG** for screenshots with text
- Use **WebP** for better performance (optional)

### Optimization
- Compress images before uploading
- Recommended tools:
  - TinyPNG (https://tinypng.com)
  - Squoosh (https://squoosh.app)
  - ImageOptim (Mac)

### Dimensions
- Hero images: 1920x1080px
- Project cards: 800x450px minimum
- Profile photo: 500x500px (square)

## Quick Setup

1. Screenshot your projects or use project mockups
2. Rename files according to the list above
3. Compress/optimize images
4. Place in this `images/` folder
5. Verify paths in index.html match filenames

## Placeholder Option

If you don't have project images yet, use a free placeholder service:
- https://via.placeholder.com/800x450/0a0e27/00d4ff?text=Project+Name
- https://source.unsplash.com/random/800x450/?technology

Or use the solid color placeholders already configured in the CSS.

## CV/Resume

Place your CV as `CV_ILAG-2025.pdf` in the main `assets/` folder (not in images/).
