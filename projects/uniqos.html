UniqOS - Custom Operating System

Built from Scratch with C# and COSMOS

C# COSMOS VMWare OS Development Kernel Programming .NET

Completed December 2023 | Team Project | Pamantasan ng Lungsod ng Maynila

Project Overview

UniqOS is a custom operating system built from scratch using C# and the COSMOS (C# Open Source Managed Operating System) development kit. Designed as an academic tool for Operating Systems (CPE 0313-1) course, this project demonstrates low-level system programming, kernel development, and OS architecture principles. UniqOS provides a command-line interface for managing student records, calculating grades, and performing system operations—all running on custom kernel code.

Key Features

  • Command Line Interface: Full CLI with multiple command support and responsive terminal
  • Security Authentication: Password-protected boot sequence preventing unauthorized access
  • Student Record Management: Add and store student information (name, ID, course)
  • Grade Calculator: Input grades and automatically calculate weighted averages
  • Student Card Generator: Display student cards with pass/fail status (passing grade: 75+)
  • Hexadecimal Converter: Convert hex values to decimal for debugging and grading
  • System Controls: Shutdown, restart, version check, and help desk commands
  • Custom Kernel: Built entirely in C# using COSMOS development kit

Code Sample: Student Management 

public class Student
{
    public string Name { get; set; }
    public int StudentID { get; set; }
    public string Course { get; set; }
    public List<int> Grades { get; set; } = new List<int>();
}

public class Kernel : Sys.Kernel
{
    private List<Student> studentRecords = new List<Student>();
    
    protected override void BeforeRun()
    {
        // Security authentication
        Console.WriteLine("Welcome to UniqOS. Please enter the password:");
        string password = "admin";
        string input = Console.ReadLine();
        
        if (input == password)
        {
            Console.Clear();
            DisplayBootLogo();
            Console.WriteLine("uniqOS successfully booted.");
            Run();
        }
        else
        {
            Console.WriteLine("Password is wrong, try again");
            BeforeRun(); // Retry authentication
        }
    }
    
    private void ListStudentsWithGrades()
    {
        foreach (var student in studentRecords)
        {
            Console.WriteLine($"Name: {student.Name}");
            Console.WriteLine($"Student ID: {student.StudentID}");
            Console.WriteLine($"Program: {student.Course}");
            
            if (student.Grades.Count > 0)
            {
                double averageGrade = student.Grades.Average();
                Console.WriteLine($"Average Grade: {averageGrade:F2}");
                
                string status = averageGrade < 75 ? "Failed" : "Passed";
                Console.WriteLine($"Status: {status}");
            }
        }
    }
}

Challenges & Solutions

Challenge: Memory Constraints

Solution: Implemented efficient data structures using Lists and optimized memory allocation. Limited student records to prevent memory overflow in the constrained VM environment.

Challenge: No Built-in Console Buffer

Solution: Designed the interface to work without scrolling capability. Implemented command-based navigation and clear output formatting to work within single-screen limitations.

Challenge: Debugging Low-Level Code

Solution: Leveraged COSMOS integrated debugger with Visual Studio. Implemented comprehensive error handling and created the hexdec tool to interpret system error codes.
