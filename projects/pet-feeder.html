IoT Smart Pet Feeder System

Full-Stack IoT Solution with Real-Time Communication

Flutter C++ ESP32 MQTT IoT Real-Time

Completed November 2025 | Duration: 2 months

Project Overview

Developed a full-stack IoT solution integrating a Flutter mobile application with dual ESP32 microcontrollers for automated pet feeding and monitoring. The system implements real-time bidirectional communication using MQTT protocol, enabling remote control, manual dispensing, and live storage tracking.

Key Features

  • Flutter Mobile App: Cross-platform application for iOS and Android with intuitive UI
  • Dual ESP32 Setup: Master controller for feeding logic, slave for sensor integration
  • Real-Time Scheduling: Automated feeding schedule with customizable meal times
  • Manual Dispensing: On-demand food dispensing via mobile interface
  • Live Storage Tracking: Real-time monitoring using Load Cell and Flow Sensor
  • Water Dispensing: Flow-based water dispensing system with relay control
  • ESP32-CAM Integration: Live video streaming for remote pet monitoring
  • Non-Volatile Storage: DS3231 RTC and NVS for reliable data persistence

Code Sample: MQTT Communication 

// ESP32 MQTT Callback Handler
void mqttCallback(char* topic, byte* payload, unsigned int length) {
    String message = "";
    for (int i = 0; i < length; i++) {
        message += (char)payload[i];
    }
    
    // Parse JSON command
    DynamicJsonDocument doc(1024);
    deserializeJson(doc, message);
    
    String command = doc["command"];
    
    if (command == "feed_now") {
        int amount = doc["amount"];
        triggerFeeding(amount);
    }
    else if (command == "update_schedule") {
        updateSchedule(doc["schedules"]);
        saveToNVS();
    }
    else if (command == "dispense_water") {
        activateWaterPump(doc["duration"]);
    }
    
    // Send acknowledgment
    publishStatus("command_received");
}

// Flutter MQTT Publisher
void sendFeedCommand(int amount) {
  final payload = jsonEncode({
    'command': 'feed_now',
    'amount': amount,
    'timestamp': DateTime.now().toIso8601String()
  });
  
  mqttClient.publishMessage(
    'pet_feeder/commands',
    MqttQos.exactlyOnce,
    MqttClientPayloadBuilder()
      .addString(payload)
      .payload
  );
}

Challenges & Solutions

Challenge: Network Failure Handling

Solution: Implemented offline scheduling system using DS3231 RTC and NVS storage. The device continues feeding operations even without network connectivity, with automatic synchronization when connection is restored.

Challenge: Real-Time Weight Accuracy

Solution: Calibrated Load Cell with HX711 amplifier, implemented moving average filter to reduce noise, and used tare function for accurate measurements.

Challenge: Power Management

Solution: Implemented deep sleep mode during idle periods, optimized WiFi connection cycles, and used efficient servo control algorithms to minimize power consumption.
